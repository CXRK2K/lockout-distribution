# LOCKOUT! Public Distribution

Welcome to the 5-minute setup guide for **LOCKOUT!**, the premier trivia and academic bowl competition system. 

## Live URL & Guest Mode

Zero-setup demo access is available right now via our browser-based version:

1. **Visit our Live URL:** [https://cxrk2k.github.io/lockout-distribution/](https://cxrk2k.github.io/lockout-distribution/)
2. **Access Guest Mode:** Click "Continue as Guest" directly on the main screen to immediately join as a Spectator and test out the live trivia features without entering any registration details.

## Desktop Installer Setup

For the complete Offline-First Native Desktop experience:

1. Download the latest installer for your OS from the [Releases page](https://github.com/CXRK2K/lockout-distribution/releases/latest).
2. Run the executable and follow the standard prompt to install.
3. Upon opening the application for the first time, either Sign Up for a Full Account or start in Guest Mode.
